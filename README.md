
# The b64.tools collection

![CC-BY-NC-SA-4.0](https://licensebuttons.net/l/by-nc-sa/4.0/88x31.png)

This is a website featuring a set of tools for converting Base64-encoded Data URIs to and from image formats (including SVG), and other fun tools.

It was developed by Timothy Meade and made available to the public at [https://b64.tools](https://b64.tools).

The website contents and source code is under the [CC-BY-NC-SA-4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/) license. Visit that page for more information.

The framework used is the awesome [Mantine](https://mantine.dev/).

The site can be deployed to AWS using the Terraform scripts in the [b64tools-terraform](https://github.com/tmzt/b64tools-terraform) repo.
